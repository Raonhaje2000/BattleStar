## BattleStar

SF 요소가 들어간 우주 배경의 ARPG 게임입니다.

주인공이 함선을 타고 우주의 각 행성을 돌아다니며 우주의 치안 및 질서를 방해하는 원인을 제거하는 것이 주목적입니다.


![Battle Star ](https://github.com/Raonhaje2000/BattleStar/assets/81238355/a8dac159-9e89-4a22-90a9-048ea7cc2831)
